send(feedbackType, feedbackText, confirmSuccessWithinopt, confirmSuccessFnopt)

A high-level utility function for quickly submitting feedback. Prepares, writes, and submits feedback. If confirmSuccessWithin is defined, a confirmation of submission success must appear within confirmSuccessWithin milliseconds. If confirmSuccessFn is undefined, the default behavior will look for an rxNotify success message. Otherwise, confirmSuccessFn will be attempted until it yields a truthy value, using Protractor's wait function.
