rxInfix
directive in module elements

Description

Structural element directive used to wrap content to be placed inline with a form control element.

Display:
inline block (only as wide as necessary for content)
Parent:
rxInput
Siblings:
  • rxPrefix
  • rxSuffix
  • Any HTML Element
Children:
Any HTML Element

Usage

as element:
<rx-infix>
</rx-infix>

Example

...
<form rx-form>
  <rx-form-section>
    <rx-field>
      <rx-field-name>Time of Day:</rx-field-name>
      <rx-field-content>
        <rx-input>
          <input type="number" name="hours" />
          <rx-infix>:</rx-infix>
          <input type="number" name="minutes" />
        </rx-input>
      </rx-field-content>
    </rx-field>
  </rx-form-section>
</form>
...
