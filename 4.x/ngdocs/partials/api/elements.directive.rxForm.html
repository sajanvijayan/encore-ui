rxForm
directive in module elements

Description

The elements directive is an attribute directive meant to be used for hierarchical validation of form-related elements. This directive may be placed on ANY DOM element, not just <form>.

Display:
block (full width of parent)
Parent:
Any HTML Element
Siblings:
Any HTML Element
Children:
  • rxFormSection
  • Any HTML Element

Usage

as attribute
<ANY rx-form>
   ...
</ANY>

Example

...
<form rx-form><!-- you can use a DIV, if desired -->
  <rx-form-section>
    ...
  </rx-form-section>
</form>
...
