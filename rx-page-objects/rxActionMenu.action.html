openModal(customFunctionality) → {rxModalAction}

Returns a modal object to manipulate later, with given customFunctionality. See rxModalAction.initialize for more information about what customFunctionality means here. Using a modal object is the default action since many instances of the action menu serve to launch modals. If you're not using rxActionMenu to launch modals, over-ride this entire section when calling rxActionMenu.initialize, where you can pass in a custom actionConstructorFn.
