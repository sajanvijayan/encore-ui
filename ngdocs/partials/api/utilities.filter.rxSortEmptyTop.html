rxSortEmptyTop
filter in module utilities

Description

Filter that moves rows with an empty predicate to the top of the column in ascending order, and to the bottom in descending order.

Usage

In HTML Template Binding

{{ rxSortEmptyTop_expression | rxSortEmptyTop }}

In JavaScript

$filter('rxSortEmptyTop')()

Example

Empty Sort
var emptySort = [
    { name: { firstName: 'Adam' } },
    { }
];
emptySort | rxSortEmptyTop 'name.firstName':false
Will sort as [{}, { name: { firstName: 'Adam' } }].

Null Sort
var nullSort = [
    { name: { firstName: 'Adam' } },
    { name: { firstName: null }
];
nullSort | rxSortEmptyTop 'name.firstName':true
Will sort as [{ name: { firstName: 'Adam' } }, {}]
