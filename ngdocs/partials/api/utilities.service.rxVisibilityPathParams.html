rxVisibilityPathParams
service in module utilities

Description

Returns an object with name and method params that can be passed to rxVisibility.addMethod(). We register this by default, as it's used by the nav menu we keep in routesCdnPath.

The method is used to check if {param: 'someParamName'} is present in the current route. Use it as: 

visibility: [ 'rxPathParams', { param: 'userName' } ]
