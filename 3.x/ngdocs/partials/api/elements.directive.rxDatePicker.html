rxDatePicker
directive in module elements

Description

Basic date picker.

Notice

This element is designed to be used in conjunction with other picker elements to compose a valid ISO 8601 DateTime string in the format of YYYY-MM-DDTHH:mmZ.

rxDatePicker provides the user a 10-year range before and after the selected date, if present. Otherwise, the range is calculated from today's date.

  • This element will generate a String in the format of YYYY-MM-DD to be used as the date portion of the ISO 8601 standard DateTime string mentioned above.
  • This element will never generate anything other than a String.

Usage

as element:
<rx-date-picker
       ng-model="{Expression}">
</rx-date-picker>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelExpression

Expression that evaluates to a date string in YYYY-MM-DD format
